Snake Game

Language: C++
Date: January/2018
  • Implemented object-orientedely with a custom doubly-linked-list data structure in a class with a segment struct
  • It uses Ncurses library to provide a graphics user-interface and key-arrow listener
  • The snake grows each time it eats a fruit that has a value from 0 to 3
  • Because of the use of doubly-linked list, the time complexity for growing the snow is O(1)

Maze Game

Language: C++
Date: September/2017
  • Stuck in a huge maze, the user has to find his way out
  • The maze is generated randomly by linking together nodes
  • Each node has 6 passages to one another
  • The solutions can be provided by calling 'solution', which then searchs for the exit using Depth-First-Search
[see the code on github]

Animal Game

Language: C++
Date: April/2017
  • Similar to the 21-question game, it makes guesses and improves as it learns from the user
  • Binary tree data structure is used to take guesses and to store questions
  • Traversing the tree by the user's answer; left if no, right if yes
  • If it runs out of guesses, it asks the user for the answer and stores it in the right place in the tree
