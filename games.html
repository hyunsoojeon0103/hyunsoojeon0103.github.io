Snake Game

Language: C++
Date: January/2018
  • Implemented object-orientedely with a custom doubly-linked-list data structure in a class with a segment struct
  • It uses Ncurses library to provide a graphics user-interface and key-arrow listener
  • The snake grows each time it eats a fruit that has a value from 0 to 3
  • Because of the use of doubly-linked list, the time complexity for growing the snake is O(1)

15 Puzzle Game

Language: Java
Date: May/2017
  • Though named 15 puzzle, it can generate an N by N grid, and numbered tiles in a random order
  • A timer is installed to keeps track of the play time, which was in turn stored in a database, so that one could compete against one another

Animal Game

Language: C++
Date: April/2017
  • Similar to the 21-question game, it makes guesses and improves as it learns from the user
  • Binary tree data structure is used to take guesses and to store questions
  • Traversing the tree by the user's answer; left if no, right if yes
  • If it runs out of guesses, it asks the user for the answer and stores it in the right place in the tree
[see the code on github]

Maze Game

Language: C++
Date: September/2017
  • Stuck in a huge maze, the user has to find his way out
  • The maze is generated randomly by linking together 1000 rooms, and each room has 6 passages to another room
  • The solutions can be provided by calling 'solutions', which then searchs for the exit using Depth-First-Search
[see the code on github]

Battleships Game

Language: C++
Date: Februrary/2017
  • The ocean grid and fleets are generated in a custom class of dynamic array
  • The game is played on ruled grids where the locations of the fleets are concealed from the player
  • The number of squares for each fleet is determined by the type
  • Each fleet occupies a number of consecutive squares on the grid, arranged either horizontally or vertically
