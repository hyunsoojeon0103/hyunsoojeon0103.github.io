Sudoku Solver

Language: C++
Date: May/2019
  • It solves sudoku as a constraint satisfaction problem, a.k.a csp
  • Forward checking is used as an extension of backtracking search that employs a modest amount of propagation
  • MRV (minimal remaining values) and LCV (least constraining value) are also used as heuristics to speed up the search
  • It successfully solves the world's hardest sudoku in less than 2 seconds

N-Queen Solver

Language: C++
Date: June/2017
  • Given N chess queens on an N by N chessboard, it places the N queens in a way that does not attack each other
  • The chess queen can attack in any direction as horizontal, vertical and diagonal way
  • It returns all the possible, distinct solutions using backtracking algorithm
  • Backtracking occurs when either it can't place a chess queen due to the constraints; in this case, it backtracks to the previous stage
